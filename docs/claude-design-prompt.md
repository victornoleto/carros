# Prompt Para Claude Design

Crie uma nova identidade visual e mockups para uma aplicacao chamada Carros.

A aplicacao nasceu como uma ferramenta pessoal para encontrar o carro ideal, originalmente um Mitsubishi Lancer. Ela agrega anuncios de carros usados/seminovos de sites como OLX e Webmotors, normaliza os dados e ajuda o usuario a comparar oportunidades por preco, ano, quilometragem e localizacao.

Objetivo do produto: transformar uma pesquisa cansativa em uma experiencia clara, emocional e objetiva, ajudando a pessoa a perceber rapidamente quais anuncios merecem atencao.

Publico: pessoas pesquisando carros usados, entusiastas automotivos e compradores que querem comparar muitos anuncios sem abrir varios sites manualmente.

Telas necessarias:

1. Dashboard publico
   - Filtros laterais ou superiores: ano, preco, quilometragem, estado, cidade e modelos.
   - Grafico de dispersao mostrando preco x quilometragem.
   - Agrupamento visual por marca/modelo.
   - CTA para entrar/criar conta e salvar alerta.

2. Tabela publica de anuncios
   - Lista com marca/modelo/versao, ano, preco, quilometragem, cidade/estado, provider e ultima atualizacao.
   - Acao para abrir o anuncio original.
   - Ordenacao visual e paginacao.

3. Login
   - Tela simples, confiavel e elegante.
   - Deve comunicar que pesquisar e gratis, mas alertas exigem conta.

4. Cadastro
   - Fluxo curto para criar conta.
   - Destaque para o beneficio: salvar buscas e criar alertas.

5. Meus alertas
   - Lista de alertas salvos.
   - Cada alerta mostra nome, resumo dos filtros e status ativo.
   - Acao para remover alerta.

6. Criar alerta
   - Confirma filtros atuais antes de salvar.
   - Permite nomear o alerta.

Direcao visual desejada:

- Evite visual SaaS generico.
- Busque uma identidade com personalidade automotiva, mas sem parecer concessionaria tradicional.
- Pode usar referencias de painel de instrumentos, telemetria, garagem premium, classificados modernos e mapas de oportunidade.
- Deve funcionar bem em desktop e mobile.
- Priorize leitura rapida de numeros: preco, km e ano.
- Use hierarquia forte para destacar bons negocios.

Entregaveis esperados:

- Conceito de identidade visual.
- Paleta de cores.
- Tipografia.
- Componentes principais.
- Mockups desktop e mobile das telas listadas.
- Observacoes de UX para tornar comparacao de anuncios mais rapida.
