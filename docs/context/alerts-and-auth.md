# Login E Alertas

A aplicacao possui dois modos de uso.

## Modo Free

Visitantes podem:

- Abrir o dashboard.
- Pesquisar anuncios.
- Filtrar resultados.
- Abrir links dos providers.

## Modo Logado

Usuarios autenticados podem:

- Criar alertas a partir dos filtros atuais.
- Listar seus alertas.
- Remover alertas.
- Executar acoes administrativas privadas, como banir anuncio.

## Alertas

Um alerta pertence a um usuario e salva um conjunto de filtros em JSON. A primeira versao persiste os criterios; o envio de notificacoes pode evoluir depois.
