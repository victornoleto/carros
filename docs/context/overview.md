# Visao Geral

Esta aplicacao nasceu para ajudar na compra de um carro, originalmente um Mitsubishi Lancer. Ela agrega anuncios de diferentes portais, normaliza os dados principais e permite comparar oportunidades por preco, ano, quilometragem e localizacao.

O uso principal e responder rapidamente: quais anuncios parecem bons o suficiente para abrir, comparar e acompanhar?

## Publico

- Pessoa pesquisando carros usados ou seminovos.
- Usuario que quer comparar modelos e acompanhar mudancas sem abrir varios sites manualmente.
- Inicialmente, visitantes podem pesquisar livremente; usuarios logados podem salvar alertas.

## Funcionalidades Atuais

- Dashboard com grafico de dispersao por preco e quilometragem.
- Tabela de anuncios filtravel.
- Redirecionamento para o anuncio original.
- Sincronizacao via scraping por providers.
- Historico de alteracao de preco.
- Cadastro/login e alertas salvos por usuario.
